Why Node.js?

Normal function of node.js is to send the content of a file and send over the network.

Here is how node.js does it

  1. Sends the task to computers file system
  2. waits for the next request
  3. when the computer is done with the task node return the file content

node runs asynchronous,non-blocking, asynchronous programming which is memory efficient

What can Node.js do?
  • Change content on the web page
  • Create, open, read, write, delete files on the server
  • Add, delete , change data on the databases
Getting Started with Node.js

Once node.js is installed on the system we can run out own server as follows...

let http = require('http');
http.createServer((req,res) => {
res.writeHeader(200,{'Content-type':'text/html'});
res.write('Hello there.');
res.end();
}).listen(8080);

The above code uses the http module to create a server as per the following boilerplate.

All about Node Modules

Node modules are javascript libraries

You can include already existing library as follows:

let http = require('http');

Your can create your own module as follows say 'Hello.js' as follows

exports.hello = "Hello mate!";

Now that you,ve created your module you can import it in some other js file as follows:

let hello = require('./hello.js');

Now lets use this module in our server

let http = require('http');
let hello = require('./hello.js');
http.createServer((req,res) => {
res.writeHeader(200,{'Content-type':'text/html'});
res.write(hello.hello);
res.end();
}).listen(8080);
